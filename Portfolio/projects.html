Pokemon Type Effectiveness
An interactive web app that displays the type effectivness of a selected pokemon move type. User selects the type they want to know the effectiveness of which displays a modal with the information. Clicking off the modal closes it alalowing the user to select a new type. Made using CSS, HTML and JavaScript.
Random Deck Generator
A web app that generates a random Magic The Gathering deck based off the selected set and color. The user selects a set chosen from the ten most recent as well as a mana type. This returns a randomized 60 card deck from the selected criteria. Made using CSS, HTML and Javascript utilizing the Scryfall API.
